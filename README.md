# Project Title
*Study Cafe is an all‑in‑one study hub website that increases productivity*

Study Café was created to make studying feel more fun and engaging. It brings together simple productivity tools—like a Pomodoro timer, 
a to‑do list, and built‑in music—into one cozy, café‑themed space. This project is an early foundation for a larger vision: a fully interactive, 
game‑like study experience that motivates users to stay focused.

## 🚀 Features
*   To-do List
*   Pomodoro Timer
*   Spotify Embed

## 🛠️ Technologies Used
*   [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
*   [HTML/CSS]([link](https://developer.mozilla.org/en-US/docs/Web/CSS))

## 📋 Prerequisites
This project requires [Node.js](https://nodejs.org) (which comes with npm) installed.

## ⚙️ Installation
1.  Clone the repo:
    ```bash
    git clone [https://github.com](https://github.com/almxndmilk/study-cafe.git)
    ```
2.  Install NPM packages:
    ```bash
    npm install
    ```
3.  Run the application:
    ```bash
    npm start
    ```

## 💡 Usage
Once running, you can:
* Add and manage tasks in the to‑do list
* Start Pomodoro sessions to stay focused
* Play music through the Spotify embed
